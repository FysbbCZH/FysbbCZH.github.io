C++

一、函数

Ⅰ. 定义

定义方式：

  1. 函数的数据类型是函数的返回值类型，若类型为 $void$ 则无返回值。
  2. 形参可以是空的，但圆括号必须存在。

Ⅱ. 函数的声明

方式：

若函数在调用之前定义，则无需声明。

Ⅲ. 函数的调用

① 传值调用

方式：

传值调用时，将全局变量 $x,y$ 的值赋予局部变量 $a,b$ ，虽函数内交换了 $a,b$ 的值但 $x,y$ 的值并未被影响。

② 传址调用

方式：

与①中相比，函数的形参变成了指针，此时传入局部变量 $a,b$ 的是全局变量 $x,y$ 的内存地址，进行交换时直接对内存地址里的内容造成影响，相当于交换 $x,y$ 的值。
