写出第一个C++程序

2019-11-18 编程语言 C++ 300 字
C++

$ Pascal $ 作为我参加信息学竞赛使用的语言，伴随我走过了多年的时光，但随着计算机语言的发展，它已经逐渐被淘汰。我也必须跟上步伐，学习现在普遍使用的高级语言 $ C++ $ 。

第一个程序 $ Hello \ World $

#include <iostream>

using namespace std;

int main()
{
    cout << "Hello World" << endl;
    return 0;
}

这可以算作是一个模板的 $ C++ $ 程序，首先要写出使用的头文件，之后是主函数（main），主函数在程序中一定最先开始执行。主函数如果正常结束就会返回 $ 0 $ ，否则会返回其他的值。

头文件

使用 $ C++ $ 编写程序和 $ Pascal $ 还是有很大的不同的，最为主要的就是 $ C++ $ 中的 <font color='red'>头文件 </font>，这些是调用 $ C++ $ 函数库的必须手段。

其使用方式为：

#include <头文件名称>

这里先总结一些常用的头文件：

Ⅰ. $ iostream $ :

包括函数：$ cin,cout,endl $

注意：该头文件没有定义全局命名空间，必须使用 $ using \ namespace \ std; $

Ⅱ. $cstdio $ :

包括函数：$ getchar,putchar,scanf,printf,freopen $

Ⅲ. $ cstring $ :

包括函数：$ memset $
